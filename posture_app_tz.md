# Posture Pro — Веб-приложение для анализа осанки

> **Документ для массажиста / реабилитолога / врача**

## 1️⃣ В чем суть приложения?

Это **веб-приложение для анализа осанки** с интеллектуальной камерой и хранением данных. Главная задача — сэкономить ваше время и **автоматизировать подбор упражнений** для клиентов.

### Текущая архитектура:
- 🌐 **Веб-приложение** (доступ по ссылке из Telegram бота)
- 📱 Работает на **любом устройстве с браузером** (телефон, планшет, ПК)
- 🖥️ **Бэкенд-сервер** для сохранения данных и обработки

### Возможности приложения:
- 📸 Делает **фотографии осанки**
- 🤖 Отправляет их ИИ для **анализа**
- отчет анализа осанки со списком упражнений, после генерации ИИ - отправляется врачу для корректировки.
- после утверждения врачом - отправляется клиенту через бота.
- клиент может получить список упражнений как интерактивный план на неделю с видео и таймером, при этом каждое упражение содержит отдельную ссылку на инструкцию, как его выполнять - с видео.
- 📝 Предоставляет удобный интерфейс для **корректировки комплекса упражнений**
- Контроль (Re-check). Через 2 недели приложение просит новые фото и накладывает их на старые: «Смотри, плечо поднялось на 2 градуса».

### ⚠️ Главное правило:
**Искусственный интеллект — только помощник.** Он не общается с клиентом напрямую. Финальное решение всегда остается **исключительно за вами**.

## 2️⃣ Как это работает на практике?

### Сценарий А: Клиент у вас на приеме 🏥

1. Открываете веб-приложение на компьютере или планшете (по ссылке сервера)
2. 📸 Фотографируете клиента через камеру в 4 позициях:
   - Со спины
   - С левого бока
   - С правого бока
   - Спереди
   - *(Приложение показывает зеленый уровень для правильного положения)*
3. 📝 Вводите базовые данные: пол, вес, возраст, ФИО клиента
4. 🔄 Данные и фото **сохраняются на сервер** (защищённое хранилище)
5. 🤖 ИИ анализирует фотографии и предлагает черновой список упражнений
6. ✏️ Вы корректируете список (удаляете ненужные, добавляете свои)
7. ✅ Нажимаете «Утвердить» → готовый отчет c интерактивным планом упражнений **сохраняется** и может быть отправлен клиенту через Telegram.

### Сценарий Б: Клиент удаленно (самопроверка) 🏠

1. Клиент получает от вас адрес телеграм-бота, запустив который он может получить **ссылку на веб-приложение** (телеграм бот удобен тем, что его можно будет продвигать дальше независимо от вас)
2. 📸 Открывает приложение на своем телефоне
3. 📷 **Самостоятельно** фотографирует себя в трех позициях со встроенной камерой телефона
4. 📝 Заполняет анкету: возраст, пол, вес, ФИО
5. 📤 Нажимает «Отправить на анализ»
6. 💾 Данные **отправляются на ваш сервер**
7. 📬 **Вам приходит уведомление** (или вы видите в интерфейсе врача):
   - Фото клиента (3 ракурса)
   - Его данные (возраст, пол, вес)
   - Черновой анализ от ИИ
8. ⏰ В удобное время вы просматриваете, корректируете и «одобряете»
9. ✅ Клиент **автоматически получает** готовый комплекс упражнений в виде интерактивного плана и pdf-отчета.

### Сценарий В: Отслеживание прогресса 📊

1. Клиент через **несколько недель/месяцев** повторно фотографируется
2. 📸 Загружает новые фото в приложение
3. 🔄 Система **автоматически сравнивает** "До и После"
4. 📈 Показывает **визуальные улучшения** (симметрия, выравнивание)
5. ✅ Вы видите **прогресс клиента** и можете **корректировать комплекс упражнений**

## 3️⃣ В чем преимущество приложения? 🎯

### ✅ Интеллектуальная камера
- **Уровень горизонта** — автоматически показывает, когда фотография ровная
- **Плавное отслеживание** — датчик гироскопа следит за положением телефона
- **Защита от искажений** — гарантирует качественные снимки без перекосов
- **Метки головы и стоп** — визуальные ориентиры для правильного масштаба и положения в кадре
- **Силуэт-подсказка (опционально)** — наложение контура для точного позиционирования тела
- **Педагогический эффект** — клиент осознает свою осанку и мотивируется выпрямиться

### 🤖 ИИ-помощник (в разработке)
- **Визуальная анализ** — оценивает наклон таза, симметрию плеч, треугольники талии
- **Предварительный диагноз** — на основе своей базы упражнений
- **Безопасность** — ИИ работает **только с вашей базой упражнений**, не придумывает новые

### 📊 История и сравнение
- **Архив фото** — все снимки клиента сохраняются
- **До и После** — визуальное сравнение прогресса
- **Отслеживание результатов** — докажите эффективность вашей работы
